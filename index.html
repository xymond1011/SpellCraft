 Learn and Play!
Strings

Communicate

Learn the most basic output command! This part lets players play with Strings and introduces them how the output command works. Strings can be used to print out dialogue between characters!

Variables

Assign

This part teaches how to assign values to variables. It will start with the String data type. As players progress, they will soon be able to play with other data types and do all sorts of commands with them!

Loops

Iterate Attacks

The game starts to get interesting as players encounter more commands. A loop is a Java component that iterates a command a certain amount of times. A basic application that players can do is to iterate their spells a certain nnumber of times.

Arrays

Create Your Own Special Combo

Each players' gameplay will start showing diversity once this feature is unlocked. Arrays are a collection of items. In the game, they can be used to store several spells and attacks within an array and be called upon anytime to perform a combo attack.

-- About the Game --

Learn and Play!

SpellCraft is designed to integrate gaming elements to help boost learner engagement. It mainly aims to supplement learners' usual lessons and provide a more interactive and engaging environment to practice coding. In the game, coding will be the main weapon of the players to combat against the challenges the game has to offer.

Progression System

Commands and other methods used in programming will be slowly introduced starting with the most basic output commands. Each command would have a certain function essential for a good gameplay to ensure that players will put what they have laerned into practice if they want to play

Combat System

SpellCraft has its own unique combat system. In the early stages, players will have limited options as they will start witht the basics and with few resources to work with. This is to let them familiarize themselves with the environment. In later stages, the combat system's uniqueness will start to show. By combining different methods and commands, players will be able design their own spells, thus the name SpellCraft!

Code to Survive!

Coding is an essential skill to play the game! It's fine to start with zero knowledge as the game can help you build up a foundation. However, improvement is a must to progress further as coding is the very core of the game!
